What the Model Is Trained On

The model trains on a two-dimensional toy classification dataset. Each dot is one training example with two input values, x1 and x2, and a target class of blue or red.

Input Each sample is a point: [x1, x2]. The network only sees these two numbers, not the drawing.
Target The correct answer is class 0 or class 1. Blue and red points show those labels.
Goal Learn a curved decision boundary that gives high probability to the right class for new points.
Training Backpropagation reduces binary cross-entropy loss by updating every weight and bias with gradient descent.
